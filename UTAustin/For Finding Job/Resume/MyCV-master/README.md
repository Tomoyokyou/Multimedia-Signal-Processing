# Resume Template

This is my Resume template. If you like the format, you can just change the contents of ``cv_4.tex``.

##How to compile?

**macOS**

1. Download ``TexShop`` and install it.
2. Open ``cv_4.tex`` with ``TexShop``.
3. Change the compile method to ``XeLatex``.
4. Click ``Typeset``.
